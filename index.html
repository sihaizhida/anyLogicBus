anyLogicBus readme

一个高效的轻量级服务框架

特征

anyLogicBus是一个高效的,可扩展的,轻量级服务框架,具有下列特征:

  • 服务路由，将HTTP请求分发给具体服务模块；

  • 访问控制，支持多种访问控制策略，并且可定制；

  • 插件式服务模块，提供服务规范，支持定制开发

  • 可定制的服务目录，支持多种形式的服务目录

  • 内置多种管理服务

  • 可部署成集群管理模式，分为元数据服务器，监控服务器，服务节点进行部署

  • 基于anyWebLoader自动更新框架
